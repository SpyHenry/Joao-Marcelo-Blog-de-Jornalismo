# Blog João Marcelo Aguilera

Realizei um site para que um amigo consiga armazenar seus textos.

Blog pessoal simples em HTML, CSS e JavaScript para armazenar textos de jornalismo.

## ✨ Características

- **Design minimalista** inspirado em blogs de jornalismo modernos
- **Totalmente responsivo** para mobile e desktop
- **5 seções** organizadas: Cinema, Esporte, Cultura, Política e Opinião
- **Artigos em destaque** na página inicial
- **Sem dependências** - apenas HTML, CSS e JavaScript puro

## 📱 Compatibilidade

Funciona em todos os navegadores modernos (Chrome, Firefox, Safari, Edge).
